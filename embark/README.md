# Embark Project

This project demonstrates how compile the contract with Embark framework.

## Prerequisites

- Node.js v22 (as specified in `.nvmrc`)
- npm or yarn

## Setup

1. Install the correct Node.js version:
```bash
nvm use
```

2. Install dependencies:
```bash
npm install
# or
yarn
```

3. Start the development environment:
```bash
npm run dev
# or
yarn dev
```

## Project Structure

- `contracts/` - Smart contract source files
- `config/` - Embark configuration files
- `test/` - Test files for smart contracts

## Available Commands

- `npm run build` - Build the project
- `npm run test` - Run tests
- `npm run deploy` - Deploy contracts
