# Simple ERC20 Token Example

This project demonstrates a basic ERC20 token implementation using OpenZeppelin contracts.

## Contract

The `MyToken.sol` contract is a simple ERC20 token with:
- Name: "MyToken"
- Symbol: "MTK"
- Initial supply: 1,000,000 tokens

## Setup

1. Open this folder in Remix IDE
2. Compile MyToken.sol
3. Deploy using the desired network
